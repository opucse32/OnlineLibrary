To run the project Correctly, please follow the steps below.

*** At first you need 'xampp' the most popular PHP development environment for projects.
-> XAMPP download link ( https://www.apachefriends.org ).

Step 1 : Open XAMPP control panel.
Step 2 : Start apache & mysql from the control panel.
Step 3 : Copy this project to C:\xampp\htdocs\OnlineLibrary
Step 4 : Open your browswer (Chrome/Firefox/Opera/UC).
Step 5 : Goto adress bar & type create.php (Enter).
Step 6 : Again go to address bar & type http://localhost/phpmyadmin (Enter).
Step 7 : Now create a new database named library
Step 8 : Import library.sql file from the project.
Step 9 : Goto address bar & type http://localhost/onlinelibrary/public

	Done :-)
